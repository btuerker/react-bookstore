## Installation Guide
### Prerequisities
System dependencies:
```
$ node -v
  v10.15.3

$ npm -v
  6.4.1

```

#### Local setup
##### Clone project
```
$ git clone https://github.com/btuerker/react-bookstore.git
$ cd ./react-bookstore
```
##### Install dependencies
```
$ npm install
```
##### Build project
```
$ npm run build
```
##### Running on development server
```
# npm start
```
Now you can visit localhost:3000 with your best browser.
